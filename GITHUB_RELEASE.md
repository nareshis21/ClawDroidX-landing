# ClawDroidX Public Beta Release (v0.1.0-beta)

## Release Title: The Native AI Agent Foundation for Android

We are thrilled to announce the first public beta of **ClawDroidX**, the native agentic framework for Android. This release establishes the foundation for secure, on-device orchestration.

### Key Features
- **Native Orchestration**: Operates directly on the Android Accessibility layer—no ADB or external setups required.
- **Local UI Pruning**: Advanced computer vision algorithms scan 300+ UI nodes locally to distill the tree into the top 30 most relevant targets, keeping sensitive data off the cloud.
- **Autonomous Workflows**: Define your goals in natural language and watch the agent navigate complex apps like Uber, Paytm, and System Settings.
- **Privacy-First Design**: Sensitive operations like PIN entry and biometric triggers are handled entirely within the device's secure enclave.

### What's in this Release
- **Initial UI Tree Pruning Engine**: Integrated ONNX Runtime for local semantic analysis.
- **Multi-step Planner**: Execution paths for Uber ride discovery, device hardware control, and payment bypass heuristics.
- **Interactive Showcase**: Official website launch with viewport-aware video playback and 100% volume unlock.
- **Security Vault**: Hardened local execution paths for high-security screens.

### Getting Started
1. **Download**: Grab the `ClawDroidX-v0.1.0-beta.apk` from the assets below.
2. **Permissions**: Enable **Accessibility Services**, **Modify System Settings**, and **BT/Call Management** to allow the agent to orchestrate natively.
3. **Scale**: In settings, provide your Groq or Cloudflare keys to enable cloud-powered reasoning with our local semantic pruning.
